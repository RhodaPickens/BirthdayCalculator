#Birthday Calculator

A simple Java console application that takes in a date of birth and calculates birthday information.

## Features

- Asks the user for their birthday
- Displays the day of the week they were born
- Displays the day of the week their birthday falls on this year
- Shows today's date and the number of days until their next birthday
- Shows how old they will be on their next birthday

## How to Run

- Clone the repository
- Open in your favorite Java IDE e.g. IntelliJ or run from the terminal
- Run the app and follow the on screen prompts
